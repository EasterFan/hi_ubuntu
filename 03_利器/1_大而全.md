## WPS
广告相对较少了～～
[下载](http://linux.wps.cn/)  

#### 1. 安装
```bash
sudo dpkg -i program_name.deb

// 如果出错
sudo apt-get install -f
```


#### 2.解决 wps 字体缺失问题
在备份文件中下载字体包  

```bash
sudo unzip wps_symbol_fonts.zip
sudo cp mtextra.ttf  symbol.ttf  WEBDINGS.TTF  wingding.ttf  WINGDNG2.ttf  WINGDNG3.ttf  /usr/share/fonts
```
然后解压后的字体可以删除了
