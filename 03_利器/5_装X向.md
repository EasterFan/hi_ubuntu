> 一些很炫很酷且并没有实际用处的小东西

### hollywood 黑客帝国即时感  
```bash
sudo apt-add-repository ppa:hollywood/ppa
sudo apt-get update
sudo apt-get install hollywood
sudo apt-get install byobu
```

### oneko 小猫小狗满屏跑
```bash
sudo apt-get install oneko
# 只聚焦在顶部
oneko -tofocus
# 小女孩
oneko -tomoyo
oneko -dog
```
